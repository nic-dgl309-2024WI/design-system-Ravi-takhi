Utility classes uses for heading:

Welcome to our Website

Class name: f-f--heading

This class chooses a special font for the heading. It makes the text look different from regular text on the website.

Class name: hs-xl

This class makes the heading larger than usual, giving it a bigger size.

Class name: hs-bold

This class makes the heading text bold, making it stand out more.

Class name: line-height-heading-3

This class adjusts the spacing between lines of text in the heading. It ensures the text is not too crowded or too spread out.

letter-spacing-2

This class adds a bit of space between each letter in the heading. It helps make the text easier to read and gives it a unique style.

text-accent:

This class changes the color of the heading text to a special color used for highlighting important information. It makes the text stand out and catch the reader's attention.

Utility classes uses for body:

Explore the beauty of nature and the wonders of the world.

Class name: f-f--body

This class picks a special font style for the paragraph. It makes the text look different from other parts of the webpage.

Class name: fs-medium

This class makes the text of the paragraph a medium size, neither too big nor too small.

Class name: fw-medium

This class makes the text of the paragraph medium in weight, neither too bold nor too thin.

Class name: line-height-body-1

This class sets the space between lines in the paragraph to a certain amount. It ensures the text is spaced out nicely and easy to read.

Class name: letter-spacing-1

This class adds a little bit of space between each letter in the text. It helps make the text easier to read and gives it a unique style.

Class name: text-primary

This class changes the color of the text to a primary color chosen for the webpage. It makes the text stand out and fit with the overall design.
